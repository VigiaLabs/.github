## 🚀 The Mission
VIGIA is decentralizing road intelligence. We build high-performance, Multi-Agent Systems (MAS) that bridge the gap between compute-constrained edge hardware and scalable cloud-native infrastructure. By treating autonomous telemetry as a cohesive ecosystem rather than isolated models, we deliver real-time, actionable intelligence.

## 🧩 The Ecosystem Architecture
VIGIA is built using a strict spec-driven development philosophy, ensuring atomic reliability across our entire pipeline. 

### 1. Edge Perception (`/vigia-raspi`)
The physical ingestion layer. Optimized for ARM architectures, this system runs INT8 quantized models (YOLO, MiDaS) to extract spatial and hazard data locally, minimizing latency and bandwidth.

### 2. Cloud Infrastructure (`/vigia-amazon`)
The central nervous system. Built on AWS, this scalable backend handles real-time telemetry ingestion, fleet management, and the core routing logic for our Multi-Agent System.

### 3. Client & Operations (`/VIGIAWEB`)
The command center. A high-fidelity, real-time dashboard for visualizing road intelligence, fleet status, and system diagnostics.
